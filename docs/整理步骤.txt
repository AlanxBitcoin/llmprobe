【项目整理步骤】

由于环境限制，项目整理需要手动执行或通过以下方法之一完成：

## 方法1：运行自动化脚本（推荐）

在项目根目录打开命令行/PowerShell，执行：

```powershell
python run_migration.py
```

或快速版本：

```powershell
python quick_organize.py
```

## 方法2：手动执行（如自动脚本失败）

### 第1步：创建目录结构

在项目根目录创建以下目录：
- scripts/
- scripts/analysis/
- scripts/experiments/
- scripts/utils/
- tests/

使用CMD：
```cmd
mkdir scripts\analysis
mkdir scripts\experiments
mkdir scripts\utils
mkdir tests
```

### 第2步：创建__init__.py文件

在以下目录中创建空的 __init__.py 文件：

- scripts/analysis/__init__.py

- scripts/utils/__init__.py
- tests/__init__.py

## 验证整理成果

完成整理后，根目录应该包含：

✓ src/          （核心库 - 无变化）
✓ scripts/      （新目录，包含 analysis/, experiments/, utils/）
✓ tests/        （新目录，包含 test.py, test1.py）
✓ configs/      （配置文件 - 无变化）
✓ data/         （数据目录 - 无变化）
✓ outputs/      （输出目录 - 无变化）
✓ main.py       （主入口 - 保持在根目录）
✓ requirements.txt
✓ README.md

使用以下命令验证整理：

python main.py --help
